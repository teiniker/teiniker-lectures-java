# VS Code

To use Java effectively in VS Code, we need three core components: 
the IDE itself, a Java Development Kit (JDK), and the Extension Pack for Java.

## Extension Pack for Java

Rather than installing dozens of individual tools, Microsoft provides 
a single bundle called the **Extension Pack for Java**. 

![JavaExtensionPack](figures/JavaExtensionPack.png)

It includes:  
* **Language Support for Java™ (by Red Hat)**: Provides IntelliSense, linting, and type checking.  

* **Debugger for Java**: Allows you to set breakpoints and inspect variables.  

* **Test Runner for Java**: Supports JUnit and TestNG.  

* **Maven/Gradle for Java**: For project build management and dependency handling.
